# Vaibhav Bhardwaj - Data Analyst Portfolio

A modern, interactive, and visually stunning 3D portfolio showcasing my expertise as a **Data Analyst** and **AI/ML Professional**.

![Portfolio Preview](/public/images/preview.png)

## 🚀 Overview

This portfolio is built using **React**, **Vite**, and **Three.js** to create a high-performance, immersive user experience. It features interactive 3D elements, dynamic animations, and a responsive design tailored for data-driven storytelling.

## 🛠️ Tech Stack

- **Frontend**: React.js, TypeScript, Vite
- **3D Graphics**: Three.js, React Three Fiber, @react-three/drei
- **Animations**: GSAP (GreenSock Animation Platform)
- **Styling**: Vanilla CSS (Custom UI/UX)
- **Tools**: Python, SQL, Power BI, Excel

## ✨ Key Features

- **Interactive 3D Tech Stack**: Falling spheres representing my core competencies (Python, SQL, ML, etc.) using physics-based interactions.
- **Project Carousel**: A dedicated section showcasing my work in AI Interview Systems, Sales Analysis, and Deep Learning.
- **Education Timeline**: A clean, chronological breakdown of my academic journey and certifications.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop viewing.

## 💻 Local Development

To run this project locally, follow these steps:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/mrvaibhavbhardwaj/vaibhav-portfolio.git
   cd vaibhav-portfolio
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run the development server**:
   ```bash
   npm run dev
   ```

4. **Build for production**:
   ```bash
   npm run build
   ```
